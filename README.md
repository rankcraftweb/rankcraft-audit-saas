# RankCraft Audit

RankCraft Audit is a SaaS SEO audit tool built with Next.js, Supabase, Google Search Console, PageSpeed Insights, and client-ready reporting.

## Main Features

- Supabase email authentication
- Project-based SEO audit workflow
- Technical SEO audit runner
- Google PageSpeed Insights scoring
- Google Search Console keyword sync
- Keyword opportunity analysis
- Client-ready report page
- Compact print/export report
- Recommendations page
- Project settings with edit/delete
- Dashboard billing/pricing layout
- Public homepage and pricing page

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Supabase Auth
- Supabase Database
- Google Search Console API
- Google PageSpeed Insights API
- Vercel Deployment
